Boggle

A Word Search, but more fun because you can choose the words and the letters don't have to be in a straight line

Rules (not official, I just wrote this from memory)

  1. The word has to be longer than 2 charachters
  2. No reusing letter blocks in a word
  3. Adjacent letters in a word have to
    be adjacent on the board (including diagonals)
  4. Also, the dictionary I am using doesn't contain all words!
    in particular, I found a problem with it missing a lot of plurals, sorry!
